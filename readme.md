# Invalid Mapping Analysis

This script intakes an invalid mappings export from Oracle ARCS, separated them based on pre-defined criteria before exporting files for regional leads. The criteria can be visualized via the **Process Flow** in non_dynamic_data. 


# Files

1. Invalid mappings 
2. Profiles
3. FCCS mappings
4. Role assignment report

## To run the program

1. Open the directory in your code editor
2. Update the directory paths in **source/data_imports/import_static_data** to map the files in **non_dynamic_data** 
3. Run **main.py**
